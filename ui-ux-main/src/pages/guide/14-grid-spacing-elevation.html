Grid, Spacing & Elevation

Aturan layout, jarak antar elemen, dan kedalaman visual yang mengikuti pola halaman saat ini.

Ringkasan Grid Spacing Elevation Panduan Pakai Implementasi

Ringkasan

Aturan grid, spacing, dan elevation menjaga halaman tetap konsisten walau komponennya berbeda. Gunakan class layout untuk pola yang berulang, bukan membuat spacing baru di tiap halaman.

Grid · Stack · Gap · Shadow
Grid
Gunakan 2, 3, 4 kolom, auto-fit, sidebar grid, atau content grid sesuai kebutuhan halaman.
Spacing
Gunakan utility Tailwind scale: gap-1.5, gap-2.5, gap-4, gap-6, dan gap-8 sebagai skala utama.
Elevation
Gunakan shadow ringan untuk card, sedang untuk panel, dan besar untuk overlay/floating element.

Grid

Grid reusable berada di layout-system.css. Pola ini cocok untuk card KPI, form section, sidebar content, dan panel konfigurasi.

layout-grid is-2

Dua kolom untuk form dan panel sederhana.

layout-grid is-3

Tiga kolom untuk KPI atau ringkasan.

layout-grid is-auto

Auto-fit untuk card yang jumlahnya dinamis.

Spacing

Skala spacing utama mengikuti spacing bawaan Tailwind, tanpa bracket arbitrary value untuk margin, padding, atau gap.

Stack

gap-4 / 16pxuntuk field groupuntuk list pendek

Elevation

Gunakan elevation sesuai hirarki. Jangan memakai shadow besar untuk semua card karena halaman akan terlihat berat.

None

Area datar atau section biasa.

SM

Card kecil dan avatar group.

MD

Panel utama dan table card.

LG

Drawer, modal, dan floating menu.

Panduan Pakai

Gunakan layout class untuk pola yang berulang, lalu gunakan class halaman hanya untuk kebutuhan spesifik.

Pakai gap yang konsisten

gap-4 untuk elemen dekat, gap-6 untuk antar panel, dan gap-8 untuk section besar.

Jangan buat shadow baru per halaman

Jika butuh kedalaman visual, pilih dari token --shadow-sm, --shadow-md, atau --shadow-lg.

Implementasi

Class layout memakai utility Tailwind scale di src/css/design-system/layout-system.css. Token shadow berada di tokens.css.
