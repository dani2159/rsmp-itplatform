Panduan Pakai

Gunakan badge untuk label/status singkat dan alert untuk pesan yang perlu dibaca. Hindari membuat warna manual di luar token.

Pakai badge untuk status singkat.

Contoh: badge badge-md badge-success untuk status aktif.

Pakai alert untuk pesan yang perlu dibaca.

Contoh: alert alert-md alert-warning untuk data yang belum lengkap.

Gunakan tone semantic sesuai konteks.

Success untuk berhasil, warning untuk perhatian, danger untuk error atau risiko.

Jangan membuat warna manual per badge.

Semua tone badge dan alert harus mengikuti token semantic.

Jangan memakai alert untuk dekorasi.

Alert hanya untuk informasi yang memang perlu terlihat oleh user.

Jangan menumpuk terlalu banyak badge dalam satu baris.

Pilih status utama, sisanya letakkan di detail atau tooltip.

Implementasi CSS, Tailwind, dan React

Badge dan alert memakai CSS variable per komponen. Varian diatur lewat class seperti .badge-success atau .alert-danger.

badges.css

CSS / Tailwind v4 source

/* src/css/design-system/badges.css */
.badge {
  @apply inline-flex items-center justify-center gap-2 w-fit max-w-full whitespace-nowrap border border-[color:var(--badge-border)] rounded-full bg-[color:var(--badge-bg)] text-[color:var(--badge-color)] text-body-sm leading-none font-semibold py-2 px-3;
  --badge-bg: var(--surface);
  --badge-color: var(--slate-dark);
  --badge-border: var(--slate-tint-60);
  --badge-dot: var(--slate);
}
.badge-success { --badge-bg: var(--success-light); --badge-color: var(--success-dark); --badge-border: var(--primary-green-tint-80); --badge-dot: var(--primary-green); }
.badge-danger { --badge-bg: var(--danger-light); --badge-color: var(--danger-dark); --badge-border: var(--danger-border-soft); --badge-dot: var(--danger); }
.badge-outline { @apply bg-[color:var(--surface)] border-[color:var(--badge-dot)] text-[color:var(--badge-color)]; }
.badge-solid { @apply bg-[color:var(--badge-dot)] border-[color:var(--badge-dot)] text-[color:var(--white)]; }

.alert { @apply flex items-start gap-3 border border-[color:var(--alert-border)] rounded-[var(--radius-input-md)] bg-[color:var(--alert-bg)] text-[color:var(--alert-color)] p-4; }

HTML / React mapping

<span class="badge badge-success badge-sm"><span class="badge-dot"></span>Aktif</span>
<span class="badge badge-danger badge-outline">Tidak Aktif</span>

<div class="alert alert-warning alert-sm" role="alert">
  <span class="alert-icon">...</span>
  <div class="alert-body">
    <strong class="alert-title">Data belum lengkap</strong>
    <p>Lengkapi NIK dan penjamin sebelum simpan.</p>
  </div>
</div>

function Badge({ tone = 'primary', solid, children }) {
  return <span className={`badge badge-${tone} ${solid ? 'badge-solid' : ''}`}>{children}</span>
}
Catatan: Tone badge/alert mengikuti token semantic di tokens.css. Jangan buat warna manual per badge.
