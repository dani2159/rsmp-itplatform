Avatars

Komponen avatar reusable untuk user, dokter, pasien, status, dan avatar group.

Ringkasan Ukuran Varian Panduan Pakai Implementasi

Ringkasan

Avatar dipakai untuk identitas user, dokter, pasien, atau grup petugas. Gunakan initials saat foto belum tersedia, dan status dot hanya jika statusnya memang dibutuhkan.

Initials · Image · Status · Group
Initials
Gunakan 1–2 huruf dari nama yang tampil.
Image
Gunakan foto dengan object-fit cover agar tidak merusak ratio.
Status
Online, busy, atau away hanya untuk konteks real-time.

Ukuran Avatar

Ukuran disediakan dari XS sampai XL agar bisa dipakai di table, navbar, sidebar, card profil, dan halaman detail.

AUAUAUAUAU

Varian Avatar

Panduan Pakai

Avatar harus membantu mengenali orang atau tim, bukan menjadi dekorasi.

Gunakan initials yang jelas

Ambil dari nama yang sama dengan teks di sebelahnya, misalnya Superadmin menjadi AU.

Jangan campur terlalu banyak warna

Gunakan varian semantik hanya saat ada makna status atau kategori.

Implementasi

Class avatar berada di src/css/design-system/avatars.css.

HTML

<span class="avatar avatar-md">AU</span>
<span class="avatar avatar-md avatar-status is-online">DR</span>
<span class="avatar avatar-lg">
  <img src="avatar.svg" alt="Avatar dokter" />
</span>

<div class="avatar-group">
  <span class="avatar avatar-sm">AU</span>
  <span class="avatar avatar-sm avatar-green">DR</span>
  <span class="avatar-count">+4</span>
</div>

React mapping

const sizeClass = {
  xs: 'avatar-xs',
  sm: 'avatar-sm',
  md: 'avatar-md',
  lg: 'avatar-lg',
  xl: 'avatar-xl',
}

function Avatar({ initials, src, size = 'md', status }) {
  return <span className={`avatar ${sizeClass[size]} ${status ? `avatar-status is-${status}` : ''}`}>
    {src ? <img src={src} alt="" /> : initials}
  </span>
}
