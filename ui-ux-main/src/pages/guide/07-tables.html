Anatomi

Struktur wajib agar tabel konsisten di seluruh modul: rawat inap, rawat jalan, penunjang, inventory, billing, dan master data.

01

Category tabs

Menggunakan tab aktif dengan underline primary-blue. Tab dapat diberi ubah/hapus action jika kategori bisa dikelola.

02

Toolbar

Cari dokter pakai autocomplete input. Filter pakai dropdown select dari design system, versi table tanpa icon dan tanpa sub description. CTA utama selalu button primary di kanan.

03

Surface tabel

Header sortable, row divider soft, teks 14px, kode pakai JetBrains Mono untuk readability.

04

Pagination

Summary data di kiri. Per-page memakai dropdown select dan page navigation di kanan. Di tablet, pindah ke bawah dan wrap.

Responsive md

Design system menempatkan desktop ≥1366px sebagai utama, tablet ≥768px sebagai pendukung, dan mobile sebagai read-only untuk pimpinan.

Desktop / XL

Toolbar satu baris. Cari di kiri, filter + CTA di kanan. Tabel full width tanpa horizontal scroll jika ruang cukup.

  • min-width: 980px
  • Cell padding 20px 16px
  • Icon aksi 40px

Tablet / md

Toolbar boleh stack. Tabel tetap berbentuk table dan memakai horizontal scroll agar data klinis tidak terpotong.

  • @media max-width: 900px
  • Tab horizontal scroll
  • Pagination wrap

Mobile read-only

Untuk pimpinan dan monitoring. Hindari aksi destruktif. Prioritaskan search, filter, dan detail ringkas.

  • Sembunyikan aksi massal
  • CTA admin boleh disembunyikan
  • Gunakan compact row
