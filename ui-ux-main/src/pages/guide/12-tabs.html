Tabs

Jenis tab untuk filter, kategori, detail konten, dan panel konfigurasi.

Ringkasan Jenis Tab Panduan Pakai Implementasi

Ringkasan

Tab dipakai saat beberapa konten setara perlu berada dalam satu area. Gunakan tab untuk mengganti isi panel, bukan untuk navigasi halaman utama.

Line · Pill · Segmented · Card · Vertical
Line tab
Cocok untuk detail pasien, riwayat, dokumen, atau data yang perlu terasa ringan.
Pill dan segmented
Cocok untuk filter cepat, status, atau kategori yang sering diganti.
Card dan vertical
Cocok untuk pilihan besar, konfigurasi, atau konten dengan deskripsi pendek.

Jenis Tab

Setiap contoh disusun satu kolom agar struktur, state aktif, dan isi panel mudah dibaca. Klik tab untuk melihat perubahan kontennya.

Line Tab

Untuk detail data yang masih ringan, seperti profil, riwayat, dan dokumen pasien.

Profil pasien

Berisi identitas pasien, nomor rekam medis, tanggal lahir, dan kontak utama.

Riwayat kunjungan

Menampilkan kunjungan rawat jalan, rawat inap, dan catatan layanan terakhir.

Dokumen pendukung

Tempat kartu identitas, surat rujukan, hasil pemeriksaan, dan dokumen penjamin.

Pill Tab

Untuk filter cepat yang sering dipakai pada daftar data.

Semua data

Menampilkan data aktif dan nonaktif dalam satu daftar.

Data aktif

Cocok untuk tampilan operasional harian yang hanya butuh data siap digunakan.

Data nonaktif

Untuk arsip data yang masih perlu dilihat tanpa masuk ke pilihan utama.

Segmented Tab

Untuk rentang waktu atau pilihan pendek yang perlu terasa seperti kontrol tunggal.

Ringkasan hari ini

Menampilkan antrian, kunjungan, dan tindakan yang terjadi hari ini.

Ringkasan mingguan

Menampilkan tren layanan dan volume kunjungan selama tujuh hari terakhir.

Ringkasan bulanan

Dipakai untuk overview laporan dan monitoring target bulanan.

Card Tab

Untuk pilihan besar yang butuh label dan deskripsi pendek.

Rawat Jalan

Fokus pada pemilihan poli, dokter, jadwal, dan nomor antrian.

Rawat Inap

Fokus pada penempatan kamar, kelas perawatan, dan status bed.

Instalasi Gawat Darurat

Fokus pada triage, prioritas layanan, dan tindakan awal.

Vertical Tab

Untuk label yang lebih panjang atau konten panel yang perlu ruang baca lebih besar.

Data pasien

Gunakan panel ini untuk informasi utama yang paling sering dibaca petugas.

Data penjamin

Berisi jenis penjamin, nomor peserta, kelas layanan, dan status verifikasi.

Dokumen pasien

Berisi dokumen identitas, rujukan, consent form, dan berkas administrasi lain.

Tab dengan Badge

Untuk menampilkan jumlah data di tiap kategori tanpa perlu membuka semua panel.

Antrian aktif

Daftar pasien yang masih menunggu dipanggil atau dilayani.

Layanan selesai

Daftar pasien yang sudah selesai proses di unit terkait.

Butuh tindak lanjut

Daftar data yang menunggu dokumen, validasi, atau keputusan petugas.

Panduan Pakai

Tab harus membantu pengguna membandingkan konten yang masih berada dalam satu konteks.

Gunakan tab untuk konten setara

Contoh: Profil, Riwayat, Dokumen, Jadwal, dan Log Aktivitas dalam halaman detail yang sama.

Jangan pakai tab untuk menu utama

Jika kontennya pindah domain besar, gunakan sidebar atau topnav agar struktur halaman tetap jelas.

Implementasi

Style tab berada di src/css/design-system/tabs.css. Interaksi klik memakai src/js/tabs.js dan atribut data-tab-target.

HTML

<div class="tabs tabs-line" data-tabs>
  <div class="tabs-list" role="tablist">
    <button class="tab is-active" type="button" role="tab"
      aria-selected="true" data-tab-target="panel-profile">
      Profil
    </button>
    <button class="tab" type="button" role="tab"
      aria-selected="false" data-tab-target="panel-history">
      Riwayat
    </button>
  </div>
  <div class="tab-panel" id="panel-profile" role="tabpanel">...</div>
  <div class="tab-panel" id="panel-history" role="tabpanel" hidden>...</div>
</div>

React mapping

const tabVariant = {
  line: 'tabs-line',
  pill: 'tabs-pill',
  segmented: 'tabs-segmented',
  card: 'tabs-card',
  vertical: 'tabs-vertical',
}

function Tabs({ variant = 'line', children }) {
  return <div className={`tabs ${tabVariant[variant]}`} data-tabs>{children}</div>
}
